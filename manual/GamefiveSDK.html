endSession(endingParams)

Defines the end of a session. A session is a continued user activity like a game match.
It should end with the score of that session. Ideally a session ends when the player cannot continue his match and must play again from the beginning.
For example: if the player has x "lifes" then the session only ends when all the x "lifes" are lost.
.startSession must be called first.
Parameters:
Name Type Description
endingParams object Some parameters can be sent inside an object to enrich the user statistics.

score object User score for the ended session.

startSession()

Defines the start of a session. A session is a continued user activity like a game match.
Ideally a session starts when the player starts playing the game from the beginning and his score is set to zero.
For example: when the players has lost all his "lifes" and clicks on "Play Again" button.
