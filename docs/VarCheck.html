(static) get(root, propList)

Traverses an object iteratively through a list of properties (accessors). Example:

var obj = { 1: { 2: { 'a': true } } };

VarCheck.get(obj, [1, 2, 'a']); // true

VarCheck.get(obj, [1, 2, 'b']); // undefined

Parameters:
Name Type Description
root Object the object to traverse
propList Array the list of properties we expect to find in the object
